About the Gloss and Analyze tabs
In Texts & Words, the Gloss tab and Analyze tab are two interlinear views to Baseline tab contents. They are basically the same except for these differences:

The Gloss tab includes the Add Words to Lexicon feature.

The lines (Word, Morphemes, Word Gloss, and so on) that appear by default are different. The Configure Interlinear lines dialog box, and various right-click menu commands allow you to change which lines are displayed in a tab independently.

The following topics help you learn the uses for these tabs:

Texts & Words overview gives an important overview of what you do with texts and the wordforms (words) in them.

Interlinear Texts overview lists all the tabs.

Parsing words overview gives an important overview for using computational parsers.

Analyze text overview lists tasks that you typically do manually.

Specify a word gloss gives steps to only add word glosses. You can specify word glosses in either the Gloss or Analyze tab, unless you want to add words to the lexicon. The Gloss tab is convenient if you will use the Text Chart tab before you fully analyze the text.
